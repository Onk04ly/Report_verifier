  ```python
  @dataclass
  class DiseaseKBBuckets:
      centroids: Dict[str, np.ndarray]       # disease_slug -> centroid vector (768,)
      article_indices: Dict[str, List[int]]  # disease_slug -> list of KB row indices
      config_snapshot: Dict[str, Any]        # snapshot of disease config at build time
  ```

  ### 3. build_disease_buckets(kb_csv_path, embeddings_path, config=None) -> DiseaseKBBuckets

  Algorithm (per D-04 and D-05):

  Step A — Load data:
  ```python
  cfg = config or get_global_config()
  dc = cfg.get_disease_config()
  kb = pd.read_csv(kb_csv_path, usecols=['category', 'quality_score'])
  embeddings = np.load(embeddings_path, mmap_mode='r').astype(np.float32)
  # Normalize all embeddings once (cosine sim = dot product on unit vectors)
  norms = np.linalg.norm(embeddings, axis=1, keepdims=True)
  norms = np.where(norms == 0, 1.0, norms)
  norm_embeddings = embeddings / norms  # shape (24731, 768)
  ```

  Step B — Compute centroids per disease:
  For each disease slug in dc['disease_list']:
    1. candidate_mask = kb['category'].isin(DISEASE_CATEGORY_KEYWORDS[disease])
    2. candidate_indices = np.where(candidate_mask)[0]
    3. If len(candidate_indices) == 0: raise ValueError(f"No KB articles for {disease}")
    4. candidate_quality = kb['quality_score'].iloc[candidate_indices].values
    5. top_k_indices = candidate_indices[np.argsort(candidate_quality)[::-1][:dc['centroid_top_k']]]
    6. centroid = norm_embeddings[top_k_indices].mean(axis=0)
    7. centroid /= (np.linalg.norm(centroid) or 1.0)   # re-normalize

  Step C — Assign all KB articles to disease buckets (cosine sim on normalized vectors = dot product):
  For each disease slug:
    sim = norm_embeddings @ centroids[disease]           # shape (24731,) — efficient batch dot
    bucket_indices = np.where(sim >= dc['centroid_sim_threshold'])[0].tolist()

  Step D — Resolve overlaps (D-05 rule: assign to disease with highest cosine sim):
  overlap_candidates = set(type1_diabetes_indices) & set(metastatic_cancer_indices)
  For each overlapping index:
    assign to the disease with higher sim[index]
    remove from the other disease's list

  Return DiseaseKBBuckets(centroids=centroids, article_indices=article_indices,
                           config_snapshot=dc)

  ### 4. save_disease_buckets(buckets, output_path) — saves centroids to .npz
  ```python
  np.savez(output_path, **{slug: vec for slug, vec in buckets.centroids.items()})
  # Also save article indices as separate arrays with suffix _indices
  np.savez(output_path,
      **{slug: vec for slug, vec in buckets.centroids.items()},
      **{f"{slug}_indices": np.array(idxs) for slug, idxs in buckets.article_indices.items()})
  ```

  ### 5. load_disease_buckets(npz_path) -> DiseaseKBBuckets
  Load from .npz; reconstruct centroids dict and article_indices dict; config_snapshot = {}.

  ### 6. CLI entrypoint:

  ```python
  if __name__ == "__main__":

  ```
